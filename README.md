# Agriculture---Chatbot
The Agriculture Chatbot is a simple web-based application that helps farmers by providing information on crops, pests, fertilizers, and irrigation. It uses basic AI logic and supports voice interaction for easy communication. The system is lightweight, user-friendly, and can be converted into a mobile app.HTML is used for structure, and CSS is used for designing the user interface.JavaScript handles the chatbot logic, user interaction, and voice features.

## Futures
Chatbot for agriculture queries 
Provides crop, pest, fertilizer suggestions
Voice input and voice response 
Simple farmer database 
Easy to use and works offline

## Technologies Used
HTML – Creates the structure of the chatbot
CSS – Designs the user interface
JavaScript – Handles chatbot logic and interactions
Web Speech API – Enables voice input and output
Local JS File – Stores farmer database
